# 🏦 Smart Bank Automation System

A professional Banking Automation Testing Project developed using **Java, Selenium WebDriver, TestNG, Maven, HTML, CSS, and JavaScript**.

This project simulates a modern banking application with a modern dark-themed UI, account balance management, fund transfer functionality, transaction history tracking, and automated testing using Selenium WebDriver.

## 📌 Project Overview

The Smart Bank Automation System is designed to automate banking workflows using Selenium and TestNG. The application includes login authentication, account balance display, fund transfer, transaction statements, and automated testing with reporting.

### 🔐 Login Module

## 🚀 How to Run

### Step 1: Clone Repository

```bash
git clone https://github.com/swathi-murali05/BankingAutomation.git
```

### Step 2: Open Project

Open the project in:

- Visual Studio Code
- Eclipse IDE
- IntelliJ IDEA

### Step 3: Install Dependencies

```bash
mvn clean install
```

### Step 4: Execute Tests

```bash
mvn test
```
